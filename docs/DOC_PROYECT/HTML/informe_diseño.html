    6. Perspectiva del Producto

      Las perspectivas del producto son la elaboración de un aplicativo web, el cual va a estar en capacidad de llevar el registro de los usuarios, la asistencia de cada alumno, las notas de dichos estudiantes y el control de pago de mensualidades

    7. Funciones del Producto

      Las funciones con las que contará el sistema son:
      • Registro de usuarios:
        Usuario Administrativo
        Usuario para docentes
        Usuario para alumnos/padres de familia
      • Iniciar sesión con el documento de identidad y contraseña
      • Menú principal, con diferentes funciones, dependiendo del rol del usuario
      • Registro de docente, que a su vez permitirá inhabilitarlos cuando ya no haga parte de la institución
      • Registro de asistencia de cada alumno.
      • Registro de notas
      • Realización de boletines

    8. Características de Usuario

      El sistema cuenta con tres tipos de usuarios:
      • El primero usuario se le atribuye al personal directivo, en este caso a la directora, con un nivel de educación superior, tener por lo menos, un nivel intermedio del uso de la informática.
      Las funciones que cumple este usuario son:
      • Registrar a los docentes y a su vez poder inhabilitarlos si ya no hacen parte de la institución.
      • Llevar el control de los pagos de mensualidades de cada alumno
      • El siguiente corresponde a los docentes, quienes deben tener un nivel de educación superior y conocimientos básicos acerca de computación
      Las funciones que tienen son:
      • Registrar la asistencia de cada alumno
      • Ingresar las notas a los estudiantes
      • Realizar los boletines correspondientes de cada niño
      • El ultimo usuario son los alumnos/padres de familia, los cuales quizá no tengan el conocimiento acerca de cómo usarlo
      • Este usuario tendrá la posibilidad de ver su progreso/nivel académico

      mostrar la información en un determinado tiempo (lo más pronto posible)
